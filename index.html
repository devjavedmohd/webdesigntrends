Designed for Salesforce

Provar is designed to test all your Salesforce functionality including validation rules, triggers and permission sets for all your users. Provar supports SFDC Communities, VisualForce, Single Sign On and Standard layouts using your browser of choice.

The only test tool which uses metadata to generate test steps and can also clean-up data at the end of your test. Provar will validate all your tests against metadata so when objects change it will report on tests are no longer working.

Seamless UI and API Testing

An automated testing suite which solely relies upon UI testing will be difficult to maintain and slow to execute. Provar also provides drag and drop APIs to create and assert your Salesforce objects and provides all the field types including picklists making it easy to test with valid data.

Provar understands and can test Salesforce UI pages without the need for creating and maintaining mapping files. It can also test complex Visualforce pages with embedded JQuery using its unique solutions.

Enterprise Ready

Achieve End to End automation and test the Integrations for your project. Provar supports testing with Databases, Messaging systems, Web services and Email. Trigger execution of your full regression pack from your Continuous Integration tool of choice and receive results including screenshots in Html, Pdf or Excel.

No programming skills required

Provar has a unique test authoring process which means no coding skills are necessary to author tests, saving time and improving readability for all. It supports best practices such as modularity, readability and abstraction which reduces test maintenance effort. It provides drag and drop test authoring and functionality to debug your tests and manage multiple environments.

Use a Database or Excel to create Data Driven tests and combine this with our in-built APIs to perform repetitive tasks such as loading large volumes of data or migration tasks.

Quick ROI

Provar is both easy to use and affordable. Investing in your regression will allow you to release last minute fixes with confidence and is a key enabler for agile projects.

Provar is designed to ensure future test maintenance is quick and effective. This will allow the team to be increasing test coverage rather than fixing the regression pack each Iteration.

In contrast to other commercial tools, Provar does not charge for additional plug-ins.
